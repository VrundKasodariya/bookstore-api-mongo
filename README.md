# 📚 Bookstore API

This is a simple Express + MongoDB REST API that allows CRUD operations on a collection of books.

## 🚀 Features

- Get all books
- Get a book by ID
- Add a new book
- Update an existing book
- Delete a book

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- MongoDB Atlas
- Nodemon
